# Google SEO Fundamentals & Search Guidelines

Derived from Google Search Essentials (formerly Webmaster Guidelines) and the SEO Starter Guide.

## Core Pillars of Google Search

### 1. Crawlability & Indexability
- **Robots.txt**: Ensure critical pages aren't blocked.
- **Sitemap.xml**: Provide a clear map of all relevant URLs.
- **Canonicalization**: Use `rel="canonical"` to prevent duplicate content issues.
- **URL Structure**: Simple, descriptive, and human-readable.

### 2. Content & Quality (E-E-A-T)
- **Experience**: Does the content show first-hand knowledge?
- **Expertise**: Is the author/site an expert in the subject?
- **Authoritativeness**: Is the site a "go-to" source for this topic?
- **Trustworthiness**: Accuracy, transparency, and security (HTTPS).

### 3. User Experience (Page Experience)
- **Core Web Vitals**:
    - **LCP (Largest Contentful Paint)**: < 2.5s.
    - **INP (Interaction to Next Paint)**: < 200ms.
    - **CLS (Cumulative Layout Shift)**: < 0.1.
- **Mobile Friendliness**: Responsive design is mandatory.
- **HTTPS**: Security is a lightweight ranking signal.
- **No Intrusive Interstitials**: Pop-ups shouldn't block the main content experience.

## Snippet Optimization
- **Titles**: 50-60 characters. Place keywords first.
- **Meta Descriptions**: 150-160 characters. Include a CTA.
- **Structured Data**: Use JSON-LD to qualify for Rich Results (FAQs, Reviews, Recipes).
