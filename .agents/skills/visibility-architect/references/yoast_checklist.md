# Yoast SEO Optimization Checklist

A comprehensive checklist for on-page SEO and readability based on Yoast's criteria.

## 1. On-Page SEO Analysis
- [ ] **Keyphrase**: Is the target keyphrase present in the page title?
- [ ] **Keyphrase Density**: Is the keyphrase used throughout the text (without keyword stuffing)?
- [ ] **Keyphrase in Intro**: Does the keyphrase appear in the first paragraph?
- [ ] **Keyphrase in Subheadings**: Is the keyphrase or its synonyms used in H2 or H3 tags?
- [ ] **Keyphrase in Meta Description**: Does the meta description contain the target keyphrase?
- [ ] **Keyphrase in Image Alt**: Is the keyphrase present in at least one image Alt text?
- [ ] **Slug**: Does the URL contain the target keyphrase?
- [ ] **Outbound Links**: Does the page link to external authoritative sources?
- [ ] **Internal Links**: Does the page link to other relevant content on the same site?

## 2. Readability Analysis
- [ ] **Sentence Length**: Are sentences under 20 words? (Goal: < 25% exceed 20 words).
- [ ] **Paragraph Length**: Are paragraphs short and digestible?
- [ ] **Passive Voice**: Is the use of passive voice minimized? (Goal: < 10%).
- [ ] **Transition Words**: Do you use transition words like "however", "therefore", "because"? (Goal: > 30%).
- [ ] **Consecutive Sentences**: Avoid starting three or more sentences with the same word.
- [ ] **Subheading Distribution**: Are there subheadings every 300 words?
- [ ] **Flesch Reading Ease**: Is the text easy to read for the target audience?

## 3. SEO Metadata Quality
- [ ] **SEO Title Width**: Is the title the correct length (not too short, not cut off)?
- [ ] **Meta Description Length**: Is it between 120 and 156 characters?
- [ ] **Schema Integration**: Does the page have appropriate JSON-LD (WebPage, Article, etc.)?
